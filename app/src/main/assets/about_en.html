Application to play songs in playlists

Main goal is to create multiple playlists ( grouped within presets) and allow to quickly switch between them by single tap

When playlist is tapped, app will save current song position and fade into new song ( fade time can be configured )

Let's imagine you want to organize D&D roleplay session. You can configure couple playlists like "tavern", "dungeons" , "woods" "battle" etc. etc.

Add songs based on theme to each of those playlist, and start your session

Your party ventures into woods? You have playlist for this!

They were ambushed by some orcs and battle is starting, switch to "battle" and in matters of seconds , your theme music changes.

Once enemy is slain , just tap back to woods playlist and continue with your quest just where you finished...

How to use?

Presets

After first launch there is nothing there. You have to start by creating a preset.
It will be one of many collections of playlists, grouped together for various occasions. Thanks to this , you can have some peaceful collection for board games with family , and another with some heavy metal for rpg session with your friends ( or other way around ...)

Once you create and make preset active , new playlist is created will be assigned to it

To switch to new preset , either click your current preset name at the top of application , or choose "presets" from application menu at the bottom

it's possible to save preset contents in form of text file. Choose download icon next to trash can of preset, choose file name ( preset name by default ). All playlists and songs inside will be writen in form of list in that file

Playlists

Main application functionality, holding list of songs, and ensuring only one is active at the time

Creating playlist

To create new playlist , make sure proper preset is activated and select "Add new playlist" from application at bottom of screen
You will be asked to give name to playlist, and then be moved to edition screen

On it you can add new songs by clicking "Add new song" button. After that action new window will be open with default android file selection. Choose songs which should be added to playlist and confirm your selection

To remove song , press and hold it for a bit , which will activate multiple selection. Select all songs that you would like to remove and confirm with pressing "Remove selected songs"

Press back button on your phone , or tap back arrow at the top of the screen

Editing playlists

To edit existing playlist choose "Edit playlist" from context menu available under three dots on playlist card

Edition screen is identical to creation screen and works exactly the same

Changing playlist card look

It's possible to change unique look of each playlist separately. To change look, select "Change playlist look" from context menu available under three dots on playlist card

On this screen you can pick and crop photo or picture from your gallery , by clicking "Change background" button

If you would like to remove existing background and restore default playlist card look, select "Remove background" button

To adjust text visible on picture, select one of available colors by clicking "change text color" button. You can additionally add shadow outline to improve it's readability ( true by default )

Removing playlists

To remove playlist, select "Delete playlist" from context menu available under three dots on playlist card
Please note that this operation cannot be undone!.

No physical files will be removed during that operation

Coping playlist

It's possible to duplicate your previously created list in order to make more of similar ones, or have exact copy within other preset

To copy playlist , select "Copy playlist" from playlist context menu. This will show a toast message with information that playlist is now in application clipboard. Now select other preset , or stay in current and then select "Paste playlist" from bottom application menu

This operation will create exact copy of playlist , including all songs information, background, text color etc. Only current song and, if playlist is currently active is not carried over. Once this operation is done, app clipboard will be cleared and paste option won't be available until next playlist is copied

Please note that once playlist is copied and pasted it's not linked to original playlist. New playlist is just a duplicate and changing any of them is not affecting other

Changing order of playlist

It's possible to change order of playlist within preset.
To do so long press on playlist and then drag and drop it to new place. New order will be saved right away. Any new added or pasted playlist is always added at the last position

Playback

To start playing your playlist , all you need to do is to either click it and activate it , or press play button if it's already active. Play button is available at the bottom of the screen together with other multimedia buttons

If for some reason application cannot play one of songs ( bad format or file was moved/deleted ) application will show notification , remove song from that playlist and attempt to play next song from playlist

Shuffle playlist

By default application will try to play songs in random order from selected playlist. It will be done by temporally creating random list of all songs from playlist and going through it one by one. When list will play last song, new one will be randomly created and current song will be on last position

Random order is generated every time playlist is activated, which means that if we switch playlists often , and playlist is not big enough , some songs might play more often

Settings

Application has some basic settings

Dark mode

How application should be displayed, light mode by default

Fade between songs duration

How long ( in seconds ) should the transition be when moving from one song to another, or when activating second playlist

Don't turn off screen

When selected , application will prevent from turning off screen as long as it's active

Troubleshooting

If something goes wrong and application crashes, please create issue on github

In settings , tap version number 5 times to enable developer mode. Then you can enable debug log. This will store some debug information on user interaction on application

Whenever application crashes, logs files are generated. Please include those to issue

Logs are created in Android/data/pl.qprogramming.themplay/files/logs
